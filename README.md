# SaladGo - Fresh Salads & Vegetables Delivery

A modern, fast, and responsive website for SaladGo - delivering fresh salads, vegetables, sprouts, and healthy products quickly to your doorstep.

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **JavaScript**: ES6+
- **Package Manager**: npm or yarn

## Features

✅ Modern responsive design
✅ Smooth scrolling navigation
✅ Product showcase with hover effects
✅ Subscription plans section
✅ Newsletter subscription
✅ Fast delivery highlights
✅ Mobile-friendly layout
✅ Beautiful gradient backgrounds
✅ Image optimization ready

## Installation

1. **Clone or navigate to the project directory**
   ```bash
   cd SALADGO
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```
   
   Or with yarn:
   ```bash
   yarn install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
   
   Or with yarn:
   ```bash
   yarn dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## Customization

### Change Colors
Edit `tailwind.config.js` to customize the color scheme.

### Update Images
Replace image URLs in `app/page.js` with your own images.

### Modify Content
Edit text, prices, and features directly in the component.

### Add New Sections
Create new sections by duplicating existing ones in `app/page.js`.
